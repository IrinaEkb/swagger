Overview

This documentation will help you discover, test, and learn about Intapp Platform API’s. The endpoints listed below are interactive and will directly modify data within your firm’s Open application, allowing you to experience their impact first-hand. Details found on expanding each endpoint aim to clarify the API user behavior by specifying contracts and also facilitating their trial to receive responses. To get you started rapidly, this site contains guides and examples for authentication, as well as steps to set up and operate both the Intake and Conflicts modules of Open.
